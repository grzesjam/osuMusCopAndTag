# osuMusCopAndTag

Simple python script to copy music from Osu! and change TagID and name of file to correct one.


## Usage

Just simply run and put path to Songs folder from Osu.
Files are created in "output" folder.



### Prerequisites

```
Python 3
```
```
Mutagen
```
